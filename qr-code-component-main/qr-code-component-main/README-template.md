### What I learned

This project helped me practice creating card-based layouts with CSS. I learned how to center elements using Flexbox and add subtle shadows for a modern look.

```css
.container {
  background: white;
  padding: 20px;
  border-radius: 20px;
  box-shadow: 0 0 20px rgba(0, 0, 0, 0.1);
  max-width: 320px;
  text-align: center;
}
```
